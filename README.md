# 入席 — 思想坐标系

> 加入一场跨越千年的对话

**入席**是一个思想探索工具：输入一个你正在思考的问题——自由、死亡、爱、意义——它会告诉你，历史上哪些思想者曾经回应过同样的问题，他们说了什么，以及为什么那样说。

搜索结果按出生年份排列，从公元前 6 世纪的老子、释迦牟尼，到 20 世纪的加缪、卡尼曼。你看到的不是一个答案，而是一场跨越 2500 年的圆桌对话。

## 核心功能

- **关键词搜索** — Google 风格搜索栏，支持中文输入法，实时匹配 45 位东西方思想者的观点
- **时间线展示** — 搜索结果按出生年份分组（先秦·古希腊 → 古典时期 → … → 20 世纪），一眼看到人类思想的演化轨迹
- **十个永恒之问** — 首页展示人类追问了几千年的经典问题，点击即可看到各家回应
- **思想者档案** — 点击任意思想者头像，查看其人生履历和思想演变历程
- **话题标签** — 15 个高频哲学关键词（自由、幸福、真理、孤独……），一键探索

## 数据库

覆盖 5 类、45 位思想者，横跨东西方文明：

| 类别 | 代表人物 |
|------|----------|
| 哲学家 | 老子、苏格拉底、柏拉图、康德、尼采、海德格尔、维特根斯坦 |
| 作家 | 苏轼、莎士比亚、陀思妥耶夫斯基、卡夫卡、鲁迅 |
| 诺贝尔奖得主 | 泰戈尔、加缪、海明威、马尔克斯、莫言、卡尼曼 |
| 近代学者 | 弗洛伊德、荣格、弗洛姆、弗兰克尔、汉娜·阿伦特 |
| 思想大成者 | 孙子、马可·奥勒留、王阳明、甘地、马克思 |

每位思想者包含：核心语录、思想洞察、人生大事年表、思想分期演变。

## 技术栈

- **React 19** + **Vite 6** — 轻量快速
- 纯前端，无后端依赖，本地即可运行
- 中文输入法安全（`compositionstart` / `compositionend` 事件处理）

## 快速开始

```bash
git clone https://github.com/Xin1896/ruxi.git
cd ruxi
npm install
npm run dev
```

打开 `http://localhost:5173` 即可使用。

## 设计理念

这个工具的出发点不是「评价谁更深刻」，而是让你看到：你此刻正在纠结的问题，两千五百年前就有人在认真思考了。

你不是在阅读哲学史，你是在**入席**——坐到这张跨越时空的圆桌旁，听听他们怎么说，然后写下你自己的回应。
